# A-STAR Logo Assets

Current production logo assets are PNG files in `public/logos/astar/`.
Existing SVG files have been moved to `public/logos/astar/legacy/`; they are
legacy/outdated and should not be used until regenerated from the new logo.

## Current Production Usage

| Usage | File | Notes |
| --- | --- | --- |
| Header mark | `astar-mark-light.png` / `astar-mark-dark.png` via `Logo` | Mark-only PNG home link |
| Hero mark | `astar-mark-light.png` / `astar-mark-dark.png` via `Logo` | Mark-only PNG with HTML headline text |
| Sidebar mark | `astar-mark-light.png` / `astar-mark-dark.png` via `Logo` | Mark-only PNG |
| Footer wordmark | `astar-horizontal-light.png` / `astar-horizontal-dark.png` via `Logo` | Horizontal production PNG |
| Stacked logo | `astar-stacked-light.png` / `astar-stacked-dark.png` | Available for non-hero brand contexts |
| Favicon | `favicon-512.png` | PNG icon metadata |
| Apple touch icon | `apple-touch-icon.png` | PNG icon metadata |
| OpenGraph/social image | `astar-og-image.png` | Static social image asset; dynamic `/opengraph-image` remains available |

## Legacy SVGs

The old SVG files are stored under `legacy/` for reference only. Do not import
or hardcode those paths in active site code.
