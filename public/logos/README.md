# Logo assets

Production A-STAR assets live in `public/logos/astar/` and are mapped through `lib/logos.ts`.

Partner and collaborator logos should live under `public/logos/partners/` with transparent backgrounds when possible.

## Guidelines

- Prefer SVG for partner logos when available.
- PNG/WebP fallbacks are fine for assets with photographic detail.
- Keep backgrounds transparent.
- Avoid baked-in padding; layout components handle spacing.
